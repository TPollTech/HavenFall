Como instalar

Instalação simples

Baixe o executável portátil, coloque em uma pasta fixa e abra. Se o Windows mostrar aviso de segurança, confirme que deseja executar o aplicativo.

  1. Baixe o arquivo HavenFall Desktop.
  2. Salve em uma pasta, por exemplo Jogos/HavenFall.
  3. Execute o arquivo baixado.
  4. Use o menu do jogo para abrir saves e logs.
